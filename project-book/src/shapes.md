# Shapes

Syntax:

```
[shape name] [shape keywords]
  [fields...]
```

## Rectangle

Define with keyword `RECT`.

- **height**: the rectangle's height.
- **width**: the rectangle's width.
- **position** <[_top_, _left_]>: top left coordinate of the rectangle.

Declaration example:

```
r1 RECT
  height 10
  width  10
  position  [0, 0]
```
