# Colors

Already existing color are defined and can be used, but it is also possible to declare them.

## Constant colors:

| Color name | keyword   |
| ---------- | --------- |
| Black      | `BLACK`   |
| Red        | `RED`     |
| Green      | `GREEN`   |
| Yellow     | `YELLOW ` |
| Blue       | `BLUE`    |
| Magenta    | `MAGENTA` |
| Cyan       | `CYAN`    |
| White      | `WHITE`   |

## RGB Color code:

Can declare rgb color code with the following function: `rgb(red, green, blue)`.

```
red   = rgb(255, 0, 0)
green = rgb(0, 255, 0)
blue  = rgb(0, 0, 255)
```

## Hexa color code:

Can declare an hexa color with the following syntax: `#FFFFFF`.

```
white = #FFFFFF
black = #000000
```
